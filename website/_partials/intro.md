**rig** is the R Installation Manager: a command line tool that installs,
removes, configures and switches between multiple versions of R on macOS,
Windows and Linux. It works without system requirements on any platform, lets
you select R versions with symbolic names like `release`, `oldrel` or `devel`,
runs multiple R versions side by side via quick links, and sets up user package
libraries, [pak](https://pak.r-lib.org) and the default CRAN mirror for you. On
macOS it also ships a menu bar app for viewing and switching the default R
version.
